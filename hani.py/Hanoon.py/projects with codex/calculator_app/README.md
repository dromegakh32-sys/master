# Project3 - Dark Neon Calculator (Python)

Basic calculator UI built with standard `tkinter` only.

## Run

```powershell
python calculator_app.py
```

## Features

- Dark Neon style interface
- Basic operations: `+`, `-`, `*`, `/`
- Decimal input with duplicate-dot protection
- Clear button (`C`)
- Safe expression evaluation with strict character filtering
- Error handling for invalid expressions and divide-by-zero

No external packages are required.
