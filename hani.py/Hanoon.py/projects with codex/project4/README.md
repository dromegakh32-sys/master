# Project4 - Neon Tic Tac Toe (Python)

Tic Tac Toe game built with Python `tkinter` only.

## Run

```powershell
python tic_tac_toe.py
```

## Features

- 2-player mode (`X` and `O`)
- Neon/dark UI style
- Win and draw detection
- Winning line highlight
- `New Round` button to restart instantly

No external packages are required.
